# Descripcion de los módulos

busqueda_tavily

    Permite buscar información de algún tema, la herramienta devuelve el contenido que encontró
    y el url de la fuente consultada.

obt_titulo

    Realiza un request a las páginas donde se solicitó la información, para obtener el títulos del
    artículo o blog

proyecto    

    Script con la lógica principal del programa, aqui se encuentra el chat diseñado con streamlit
